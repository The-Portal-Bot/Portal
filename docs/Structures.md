## Structures

* _prefix:_ __{{__
* _suffix:_ __}}__
* _(Structures are grammatical attributes to control the structure of the output)_

| Attributes | Description | Usage |
| :--------- | :--------- | :--------- |
`if` | conditional statement  | `{{ "if": "John", "is": "===", "with": "John", "yes": "same name", "no": "not the same name"}}`
